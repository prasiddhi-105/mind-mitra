# MindMitra Design System

A vibrant, empathetic, and playful design system for an AI-powered mental wellness platform.

## Visual Identity

The visual direction, "MindMitra Wellness," balances high-tech AI capabilities with human warmth. It uses soft geometry, high-energy colors, and glassmorphic layers to create a "safe sanctuary" feel.

## Colors

Our palette is designed to be uplifting and calming simultaneously.

| Name              | Hex       | Usage                                              |
| :---------------- | :-------- | :------------------------------------------------- |
| **Primary**       | `#005da7` | Primary actions, branding, and high-emphasis text. |
| **Surface**       | `#f7f9fb` | Main background color for a clean, airy feel.      |
| **Surface-Low**   | `#f2f4f6` | Secondary backgrounds and grouped content.         |
| **Accent Orange** | `#ff9e6d` | Playful CTAs and secondary highlights.             |
| **Soft Green**    | `#b8f28b` | Success states and wellness-themed sections.       |
| **Soft Purple**   | `#dcd0ff` | "Safe space" elements and secondary cards.         |

## Typography

Using **Inter** for its clarity and approachability.

- **Headlines**: Bold weights with generous tracking for a modern, friendly look.
- **Body**: Regular weight, optimized for readability with high line-height (1.6).
- **Labels**: Medium weight, all-caps for utility elements.

## Design Tokens

- **Roundness**: `ROUND_EIGHT` (large, bubbly corners for friendliness).
- **Elevation**: Subtle glassmorphism using `backdrop-blur-xl` and low-opacity borders (`white/20`).
- **Spacing**: Generous gutter padding and section vertical spacing to create "breathing room."

## Motion Principles

- **Floating**: Soft, vertical oscillation for hero elements (breath-like).
- **Reveal**: Staggered fade-in and slide-up for scroll-triggered sections.
- **Feedback**: Gentle scaling (`scale-95`) on button interaction.
